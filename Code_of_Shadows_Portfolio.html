Code of Shadows

A browser-based Python mystery game for adult beginners

Overview

Code of Shadows is an interactive, story-driven coding game built to teach Python through immersive gameplay. Designed for adult beginners, the game blends narrative mystery with real coding challenges across 10 chapters. Players write Python code to solve puzzles, access systems, and uncover a digital conspiracy—right in their web browser.

  • Platform: Web-based (HTML/CSS/JS + Skulpt for Python)
  • Tech Used: JavaScript, Skulpt, Python, HTML/CSS, localStorage
  • Built For: Beginners, educators, portfolio builders

Gameplay

You play as a former cybersecurity analyst turned rogue investigator. A missing AI researcher, corrupted files, encrypted logs—each chapter uncovers a deeper truth, and only your Python skills can unlock it.

Chapters & Learning Objectives

ChapterTitlePython Focus
1The Missing SignalVariables, print()
2Log In to TroubleConditionals (if, ==, else)
3Pattern in the Noisefor loops, filtering lists
4The Loopback Trapwhile loops, infinite loop logic
5Faulty MemoryFunctions, string replacement
6The Vault AccessorDictionaries and key access
7Disrupt the SignalBoolean logic (and, or, not)
8The Decryptor.lower(), .replace()
9The Watchdog ReportSimulated file I/O, error filtering
10Exit NodeFinal review: functions, loops, logic

Features

  • Code validation with feedback per chapter
  • Progress tracking via localStorage
  • Dark mode toggle
  • Ambient audio + sound effects
  • Player profile showing current progress
